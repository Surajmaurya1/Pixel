# Pixel - Premium Video Agency

A premium, high-end hero section for a video editing agency named 'Pixel'. This is built with React, TypeScript, Vite, and Tailwind CSS.

## Features
- **Video Background**: Full-screen, muted, auto-playing video background.
- **Floating Navbar**: A clean, white floating navigation bar with a subtle shadow and rounded corners. Includes a responsive menu and a primary CTA with a unique arrow icon within a circular housing.
- **Premium Typography**: Uses 'Barlow' for a modern sans-serif feel with tight tracking, combined with 'Instrument Serif' for an elegant, italicized serif font on the bold statement part of the headline.
- **Hero Area**: Centered minimal hero with a clear value proposition, subtext, and a pill-shaped secondary CTA with a play icon.
- **Fully Responsive**: Optimised for mobile, tablet, and desktop viewing.
- **Interactions**: Subtle hover states, transforms, and scale effects on buttons.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (Icons)

## Setup & Running Locally

1. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
